# Traveloop ✈️

Traveloop is a premium, full-stack travel planning platform designed to help users build multi-city itineraries, track budgets, discover activities, and share travel plans. 

## Project Structure
This is a monorepo containing both the frontend client and the backend API service.

- **[`/frontend`](./frontend/)**: The user interface, built with React, Vite, Tailwind CSS, and Framer Motion for a smooth, premium experience.
- **[`/Backend`](./Backend/)**: The REST API, built with Node.js, Express, and Sequelize ORM (PostgreSQL).

## Features
- 🗺️ **Smart Itineraries**: Build multi-city travel plans.
- 💵 **Budget Tracking**: Track transport, stay, and meal expenses.
- ✅ **Packing Lists**: Categorized checklists with progress tracking.
- 🌍 **City Explorer**: Discover destinations and activities.
- 🔗 **Public Sharing**: Share itineraries with friends and family.
